### Launch

*  git clone git@github.com:JordyBukondaTshibangu/I-SHOOTPICTURE--LARAVEL-PHP-APP.git
* cd I-SHOOTPICTURE--LARAVEL-PHP-APP
* npm install 
* composer install
* cp .env.example .env
* php artisan key:generate
* php artisan migrate:fresh
* php artisan storage:link
* php artisan serve
